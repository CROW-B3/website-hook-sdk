# website-hook-sdk

A lightweight JavaScript/TypeScript SDK for capturing user interactions on web pages and sending them into the Crow-B3 analytics/data pipeline.

## Installation

```bash
pnpm install @crow-b3/website-hook-sdk
```

## Development

```bash
# Install dependencies
pnpm install

# Build
pnpm build

# Lint
pnpm lint

# Format code
pnpm format
```
